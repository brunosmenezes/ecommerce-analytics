# E-commerce Analytics: Data Quality, Performance & Strategic Insights (2023–2025)

## 📌 Contexto de Negócio

Este projeto simula a análise estratégica de um e-commerce com dados corporativos realistas entre 2023 e 2025.  

A base contém ruídos comuns em ambientes reais, como inconsistências categóricas, valores ausentes, registros potencialmente duplicados e variações operacionais típicas de sistemas transacionais.

O objetivo não é apenas analisar vendas, mas demonstrar maturidade analítica ao conectar qualidade de dados, indicadores de performance e tomada de decisão estratégica.

---

## 🎯 Objetivos do Projeto

1. Realizar auditoria técnica da base de dados (Data Audit)
2. Identificar e tratar problemas estruturais (Data Cleaning)
3. Conduzir análise exploratória orientada a KPIs de negócio
4. Extrair insights estratégicos acionáveis
5. (Etapa futura) Aplicar inferência estatística para validação de hipóteses

## 🔎 Metodologia

### 1️⃣ Data Audit
- Estrutura da base
- Tipos de dados
- Valores ausentes
- Duplicados
- Inconsistências categóricas
- Anomalias numéricas

### 2️⃣ Data Cleaning
- Padronização de categorias
- Tratamento de missing values
- Correção de tipos
- Documentação das decisões

### 3️⃣ Exploratory Data Analysis (EDA)
Análise orientada a indicadores-chave:

- Receita total e mensal
- Ticket médio
- Mix de produtos
- Performance por categoria
- Sazonalidade
- Forma de pagamento vs valor de compra

### 4️⃣ Strategic Insights
- Identificação de riscos operacionais
- Oportunidades de crescimento
- Recomendações acionáveis

---

## 📊 Perguntas Chave

- Como a qualidade dos dados pode impactar decisões estratégicas?
- Existe crescimento consistente ao longo do período?
- Quais categorias impulsionam a receita?
- O ticket médio varia por meio de pagamento?
- Existem padrões sazonais relevantes?
